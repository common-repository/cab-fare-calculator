This plugin will add a taxi booking form on your WordPress website. You will be able to manage your vehicles and orders through the back end.

== Description ==

Taxi Booking for WordPress is a complete, standalone booking system for distance priced transportation services like taxi, limousine, shuttle, airport transfers, delivery etc.

[Get Taxi Booking Pro for WordPress here](https://kanev.com/products/taxi-booking-for-wordpress).

With this WordPress plugin, you can add a Taxi Booking form on your WordPress website.

See [Taxi Booking for WordPress demo](https://wordpress-513124-1628559.cloudwaysapps.com/booking-form/) here.

With the Pro version of the plugin you get multiple payment gateways, translation capability into any language, custom fields to collect more information or upsell products or services.
